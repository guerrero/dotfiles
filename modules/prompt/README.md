# Prompt

Configure [Starship][1] prompt for the fish shell. Starship is a fast, customizable, and cross-shell prompt.

## Configuration

The Starship configuration file is located at `~/.config/starship.toml` and is symlinked from `modules/prompt/settings/starship.toml`. You can customize the prompt by editing this file.
